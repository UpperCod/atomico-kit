<a-showcase/>

Component that allows rendering different views based on a selector and resizing the container

Installation

npm install "@atomico/kit"

Usage

import "@atomico/kit/a-showcase";

this component import the module defined by the src property, the module must require the following exports:

import { h } from "atomico";
export { render } from "atomico";

export default [
  {
    label: "Ejemplo 1",
    render() {
      return <h1>1</h1>;
    }
  },
  {
    label: "Ejemplo 2",
    render() {
      return <h1>2</h1>;
    }
  }
];

Where :

  1. export {render} ... : allows exporting the render engine to be used at the time of selection.
  2. export default [...: options for view selector.
