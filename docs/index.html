Hooks

use-lazy
Hook that allows the import of dynamic modules or promise resolution
use-custom-property
this hook allows you to use custom Property as hooks, these will be defined as part of the webcomponent
use-delegate-focus
These hooks allow to redirect the focus effect associated to the component a reference
use-event-move
This hooks allows you to observe the touch or mousemove event at the moment of the drag and drop action
use-media-resize
These hooks allow to observe the size of the reference and define a state based on this.
use-method
This hook allows associating a method of the scope of the webcomponent, as part of the context of the customElement.
